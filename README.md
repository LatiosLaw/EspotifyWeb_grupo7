# Espotify (Web)  

Reproductor de música en línea desarrollado en un sistema de 3 capas.  
Las tres capas son: el cliente, el administrador del front-end y el back-end.  

Cuenta con un sistema de usuarios que permite interactuar como si fuera una red social.  
Es decir, se pueden crear listas de reproducción de música públicas o privadas, ver los perfiles de otros usuarios, etc.  

Los usuarios no solo son consumidores de música, sino que también pueden ser creadores de contenido; por lo tanto, pueden subir sus canciones a la plataforma, asignándoles diferentes *tags* a los temas que suban.  

Para ver todas las funcionalidades del sistema, consulte la [documentación oficial](docu).  

El Front-End fue hecho en su mayoría con **Java**, utilizando tecnologías como **Servlets** para poder emplear este lenguaje en el lado web.  
También utilizando **Web Services** para manipular peticiones **SOAP**.
